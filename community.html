How to contribute

JellyClaw grows from real usage. Here's the fastest path from idea to merged PR.

Find or file an issue

Browse open issues tagged good-first-issue, or file one describing what's missing.

Fork and branch

Fork the repo, branch off main, and keep changes scoped to one thing.

Open a PR

Describe what changed and why. Small, focused PRs get reviewed fastest.

Iterate together

Expect review comments within a few days. Once it's green, it ships.

Roadmap

Where the project is now, and where it's headed.

v0.1

Core engine

Agent hierarchy, task scheduling, SQLite state store, hybrid Claude + Ollama routing.

v0.2

Telegram + dashboard

Bot-based reports and approvals, plus a live dashboard for watching agents in real time.

v0.3

MCP tools

Native Model Context Protocol support so agents can plug into external tools and data sources directly.

v1.0

Department packs

Pre-built agent teams for common functions — engineering, growth, ops — ready to configure and run.
