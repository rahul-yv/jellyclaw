# JellyClaw

JellyClaw is **the Local AI Office** — a multi-agent orchestration platform that runs a hierarchy of AI agents (CEO → department heads → workers) fully locally via Ollama, configured through simple YAML. Agents code, review, and research overnight, then report to your Telegram. MVP-scope integrations: Telegram, Ollama, Claude API, GitHub, SQLite. The core engine is in active development — watch this repo. This repository also hosts the site at [jellyclaw.in](https://jellyclaw.in).
