Questions, answered honestly

The four things everyone asks before their first install.

Is it really free?

Yes. MIT licensed, no paid tier, no feature gates. Your only costs are your own hardware's electricity and — if you opt in — Claude API tokens for high-stakes tasks.

What hardware do I need?

Any Linux or macOS box with 16GB+ RAM. An old desktop or a cheap VPS is plenty; a GPU makes local models faster but isn't required.

Does my code leave my machine?

Only if you route a task to Claude. Tasks handled by local Ollama models never touch the network, and you control the routing per task type.

How is this different from Claude Code or Cursor?

Those are interactive tools — they work while you type. JellyClaw is the orchestration layer that keeps working after you log off: it schedules, delegates, reviews, and reports without you at the keyboard.
